# Hardware Components

## Main Components

### ESP32
Main microcontroller for data acquisition and communication.

### SCT-013 Current Sensor
Used for electrical current monitoring.

### ZMPT101B Voltage Sensor
Used for voltage measurement.

### Breadboard
Used for initial prototyping and testing.

### Jumper Wires
Electrical connections between components.
