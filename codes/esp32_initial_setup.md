# ESP32 Initial Setup

## Objective

Configure the ESP32 microcontroller for initial development and electrical monitoring applications.

---

## Initial Goals

- Configure Arduino IDE;
- Establish communication with ESP32;
- Test serial monitor;
- Prepare environment for sensor integration.

---

## Hardware

- ESP32
- USB Cable
- Breadboard
- Jumper Wires

- MQTT Communication
- Node-RED Dashboard

- Fault diagnosis
- Smart grids
