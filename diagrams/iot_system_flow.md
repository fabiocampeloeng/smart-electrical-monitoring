# IoT System Flow

## System Architecture

Electrical Sensors → ESP32 → MQTT → Node-RED Dashboard

---

## Components

### Sensors
Responsible for electrical parameter acquisition.

### ESP32
Processes and transmits electrical data.

### MQTT
Communication protocol for real-time data transmission.

### Node-RED
Supervisory platform for visualization and monitoring.
